Question-1: What are the different ways to select an element in the DOM?
Ans:Use document.getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll, or getElementsByName to select DOM elements.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
Ans: innerHTML handles HTML content, innerText gets only visible text, and textContent retrieves all text, including hidden, as plain text.
Question-3: What is event delegation in the DOM?
Ans:Event Delegation allows a parent element to handle events for its children using event bubbling. Instead of adding listeners to each child, you add one to the parent.
Question-4: What is event bubbling in the DOM?
Ans:Event Bubbling is when an event starts from the target element and propagates up to its ancestors in the DOM.
Question-5: How do you create, add, and remove elements using JavaScript?
Create with document.createElement(), add with appendChild(), and remove with remove() or removeChild().
